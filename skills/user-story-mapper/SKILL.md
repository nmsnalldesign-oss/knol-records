Скилл для планирования работы команды.

## Что делать:
1. **Эпики**: Большие части системы.
2. **Шаги**: Путь пользователя от входа до цели.
3. **Задачи**: Конкретные тикеты (tickets), которые нужно сделать программисту.

## Задача:
Построй карту задач для реализации крупной функции, чтобы ничего не забыть.
