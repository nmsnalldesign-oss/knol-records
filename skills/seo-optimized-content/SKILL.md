Скилл для продвижения сайта в поисковиках.

## Чек-лист:
1. **Мета-теги**: Title, Description, OpenGraph (для соцсетей).
2. **Заголовки**: Строгая иерархия `<h1>` -> `<h2>` -> `<h3>`.
3. **Скорость**: Рекомендации по оптимизации загрузки.
4. **Sitemap и Robot.txt**: Проверка наличия и корректности.

## Действие:
Проанализируй HTML и предложи правки для улучшения индексации.
