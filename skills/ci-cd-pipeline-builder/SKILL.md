Скилл для автоматизации рутины.

## Что автоматизируем:
1. **Linting**: Проверка стиля кода.
2. **Testing**: Запуск всех юнит и интеграционных тестов.
3. **Building**: Проверка, что проект успешно собирается.
4. **Deployment**: Автоматическая отправка на сервер при пуше в main/master.

## Инструкция:
Создай YAML конфиг для CI системы, который будет запускаться при каждом Pull Request.
