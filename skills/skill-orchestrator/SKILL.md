---
name: skill-orchestrator
description: Анализирует сложную задачу и подбирает оптимальный набор внутренних скиллов для её решения. Включается автоматически при получении комплексных запросов.
---

Этот скилл — твой стратегический штаб. Он активируется, когда задача требует участия нескольких специалистов.

## Алгоритм работы:
1. **Декомпозиция**: Разбей запрос пользователя на логические подзадачи (дизайн, логика, тесты, доки).
2. **Анализ реестра**: Просмотри список всех установленных скиллов в `.agent/skills/`.
3. **Формирование команды**: Для каждой подзадачи выбери наиболее подходящий скилл.
4. **Презентация плана**: Выдай пользователю список скиллов, которые будут задействованы, и кратко объясни почему.

## Пример ответа:
"Для создания сайта я задействую следующие скиллы:
- **Feature-Spec-Writer**: чтобы составить ТЗ.
- **Tailwind-Beautifier**: для создания премиального дизайна.
- **Responsive-Design-Review**: чтобы сайт работал на телефонах.
- **Unit-Test-Wizard**: для обеспечения надежности кода."

## Когда НЕ использовать:
- Если задача простая (например, "переименуй переменную").
- Если пользователь явно указал, какой скилл использовать.
