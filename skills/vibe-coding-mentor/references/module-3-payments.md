## Урок 3.1: Как работают онлайн-платежи

### Схема процесса

```
[Покупатель] --> [Твой сайт] --> [Платёжная система] --> [Банк]
                                        |
                                        v
                               [Подтверждение оплаты]
                                        |
                                        v
[Доставка товара] <-- [Webhook уведомление] <-- [Твой сервер/n8n]
```

### Ключевые понятия

| Термин | Что это |
|--------|---------|
| **Эквайринг** | Приём платежей картами |
| **Комиссия** | % от платежа, который берёт система |
| **Webhook** | Уведомление о событии (оплата прошла) |
| **Чек** | Фискальный документ (нужен по закону) |
| **Возврат (refund)** | Возврат денег покупателю |

## Урок 3.3: Интеграция Продамус

### Шаг 1: Регистрация

1. Зайти на https://prodamus.ru
2. Зарегистрироваться как самозанятый
3. Подтвердить личность (паспорт + селфи)
4. Привязать карту для вывода средств

### Шаг 2: Создание продукта

1. В личном кабинете: Магазин → Товары
2. Добавить товар:
   - Название
   - Цена
   - Описание
   - Файл для доставки (если цифровой товар)

### Шаг 3: Получение кода кнопки

```html
<!-- Пример кода от Продамус -->
<script src="https://payform.ru/script/widget.js"></script>
<button onclick="Payform.open('ВАШ_ID_ПРОДУКТА')">
    Купить за 990 ₽
</button>
```

### Шаг 4: Настройка Webhook

1. Настройки → Уведомления → Webhook URL
2. Укажи URL твоего n8n или сервера
3. При успешной оплате Продамус отправит данные

Пример данных webhook:
```json
{
    "order_id": "12345",
    "product_id": "product_001",
    "customer_email": "client@mail.ru",
    "amount": 990,
    "status": "success",
    "date": "2025-01-30T15:30:00"
}
```

## Урок 3.4: Автоматическая доставка цифровых товаров

### Способ 1: Через Продамус (самый простой)

1. При создании товара загрузи файл
2. Продамус автоматически отправит ссылку после оплаты

### Способ 2: Через n8n (гибкий)

Workflow:
```
[Webhook от Продамус] 
    → [Проверить status === 'success']
    → [Отправить email с файлом/ссылкой]
    → [Записать в Google Sheets]
```

### Способ 3: Через свой сервер

```javascript
// Express.js пример обработки webhook
app.post('/webhook/prodamus', (req, res) => {
    const { order_id, customer_email, status } = req.body;
    
    if (status === 'success') {
        // Отправить товар
        sendProductByEmail(customer_email, productLink);
        
        // Записать в базу
        saveOrder(order_id, customer_email);
    }
    
    res.status(200).send('OK');
});
```

## Урок 3.5: Юридические аспекты

### Что нужно самозанятому?

1. **Статус самозанятого** — оформляется через приложение "Мой налог"
2. **Чеки** — Продамус формирует автоматически
3. **Налог** — 4% (физлица) или 6% (юрлица) — платится через "Мой налог"
4. **Договор оферты** — ОБЯЗАТЕЛЕН на сайте

### Обязательные страницы на сайте

1. **Договор оферты** — условия покупки
2. **Политика конфиденциальности** — как обрабатываете данные
3. **Пользовательское соглашение** — правила пользования сайтом

### Шаблон структуры оферты

```markdown
# Договор публичной оферты

## 1. Общие положения
Настоящий документ является официальным предложением (офертой)...

## 2. Предмет договора
Продавец обязуется передать Покупателю цифровой товар...

## 3. Цена и порядок оплаты
Цена указана на сайте...
Оплата производится через платёжную систему Продамус...

## 4. Доставка
Цифровой товар доставляется на email в течение 15 минут...

## 5. Возврат
Возврат возможен в течение 14 дней...

## 6. Ответственность сторон
...

## 7. Контакты продавца
ФИО: Иванов Иван Иванович
ИНН: 123456789012
Email: support@example.com
```

> ⚠️ **Важно**: Перед публикацией проконсультируйся с юристом или используй проверенные генераторы документов.
