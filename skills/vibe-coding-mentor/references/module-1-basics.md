## Урок 1.1: Как работает веб

### Что происходит когда ты открываешь сайт?

```
[Ты] --> [Браузер] --> [Интернет] --> [Сервер]
                                          |
                                          v
                                    HTML + CSS + JS
                                          |
                                          v
[Ты видишь страницу] <-- [Браузер рисует]
```

### Три кита веб-разработки

| Технология | Роль | Аналогия |
|------------|------|----------|
| **HTML** | Структура | Скелет человека |
| **CSS** | Внешний вид | Одежда и макияж |
| **JavaScript** | Поведение | Мышцы и нервы |

### Простейший пример

```html
<!DOCTYPE html>
<html>
<head>
    <title>Мой первый сайт</title>
    <style>
        /* Это CSS — стили */
        body {
            font-family: Arial;
            background: #f0f0f0;
        }
        h1 {
            color: blue;
        }
    </style>
</head>
<body>
    <!-- Это HTML — структура -->
    <h1>Привет, мир!</h1>
    <p>Это мой первый сайт</p>
    
    <button onclick="alert('Клик!')">Нажми меня</button>
    
    <script>
        // Это JavaScript — поведение
        console.log('Страница загружена!');
    </script>
</body>
</html>
```

## Урок 1.2: Как эффективно общаться с AI

### Плохие промпты vs Хорошие промпты

| Плохо | Хорошо |
|-------|--------|
| "Сделай сайт" | "Создай лендинг для продажи онлайн-курса по фотографии. Нужны секции: hero с заголовком, преимущества курса, отзывы, цена и кнопка покупки" |
| "Не работает" | "При нажатии на кнопку 'Отправить' ничего не происходит. Вот мой код: [код]. Ожидаю что данные отправятся на сервер" |
| "Измени цвет" | "Измени цвет заголовка h1 с синего на градиент от #FF6B6B к #4ECDC4" |

### Формула хорошего промпта

```
[Контекст] + [Задача] + [Детали] + [Ожидаемый результат]
```

**Пример:**
> Я делаю сайт для продажи своих фотографий (контекст). 
> Нужно добавить галерею на главную страницу (задача).
> Хочу 3 колонки на десктопе, 1 на мобильных. При клике — увеличение фото (детали).
> Покажи HTML и CSS код с комментариями (результат).

### Итеративный подход

```
Шаг 1: Попроси базовую версию
        ↓
Шаг 2: Посмотри результат
        ↓
Шаг 3: Уточни что изменить
        ↓
Шаг 4: Повтори пока не получишь нужное
```

## Урок 1.3: Настройка рабочего окружения

### Что тебе понадобится

1. **Текстовый редактор** — VS Code (бесплатный)
   - Скачай: https://code.visualstudio.com/
   
2. **Браузер** — Chrome или Edge
   - Инструменты разработчика: F12
   
3. **Node.js** — для запуска проектов
   - Скачай: https://nodejs.org/
   - Выбирай LTS версию

4. **Git** — система контроля версий
   - Скачай: https://git-scm.com/

### Проверка установки

Открой терминал (PowerShell) и выполни:

```powershell
# Проверить Node.js
node --version
# Должно показать v20.x.x или выше

# Проверить npm
npm --version
# Должно показать 10.x.x или выше

# Проверить Git
git --version
# Должно показать git version 2.x.x
```

### Расширения VS Code (рекомендуемые)

- **Live Server** — автоперезагрузка страницы
- **Prettier** — форматирование кода
- **Russian Language Pack** — русский интерфейс

## Урок 1.4: Твой первый сайт за 30 минут

### Практическое задание

Создай простой сайт-визитку:

**Требования:**
- Заголовок с твоим именем
- Фото или аватар
- Краткое описание (кто ты)
- 3 ссылки на соцсети
- Красивый дизайн (градиенты, тени)

**Как делать:**
1. Создай папку `my-first-site`
2. Создай файл `index.html`
3. Попроси AI помочь с кодом
4. Открой файл в браузере

### Шаблон промпта для задания

> Помоги создать сайт-визитку. 
> Имя: [твоё имя]
> Описание: [кто ты, чем занимаешься]
> Соцсети: [список ссылок]
> 
> Стиль: современный, тёмная тема, с градиентами и анимациями при наведении.
> Покажи полный HTML+CSS код в одном файле.
