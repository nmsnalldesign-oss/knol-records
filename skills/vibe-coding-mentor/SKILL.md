# Vibe Coding Mentor — Твой наставник по вайб-программированию

## Что такое Vibe Coding?

**Вайб-кодинг** (vibe coding) — это подход к программированию, где ты описываешь что хочешь получить на естественном языке, а AI пишет код за тебя. Ты задаёшь направление ("вайб"), AI делает техническую работу.

### Ключевые принципы:
1. **Описывай ЧТО, а не КАК** — "Сделай кнопку, которая отправляет форму" вместо "напиши addEventListener"
2. **Итерируй маленькими шагами** — Одна фича за раз
3. **Проверяй результат** — Запускай, смотри, корректируй
4. **Задавай вопросы** — Если непонятно, спроси AI объяснить

## Глоссарий терминов

| Термин | Объяснение |
|--------|------------|
| **HTML** | Скелет страницы (заголовки, текст, кнопки) |
| **CSS** | Внешний вид (цвета, шрифты, расположение) |
| **JavaScript** | Поведение (клики, анимации, отправка данных) |
| **React** | Библиотека для создания интерактивных интерфейсов |
| **Node.js** | Позволяет запускать JavaScript на сервере |
| **npm** | Менеджер пакетов (установка библиотек) |
| **n8n** | Инструмент автоматизации без кода |
| **API** | Способ общения программ между собой |
| **Хостинг** | Сервер, где живёт твой сайт |
| **Домен** | Адрес сайта (example.com) |
| **PWA** | Веб-приложение, которое работает как нативное |
