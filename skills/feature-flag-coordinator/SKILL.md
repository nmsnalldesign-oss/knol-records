Скилл для управления выпуском фич.

## Что такое Feature Flags:
- Возможность включать/выключать новую логику без деплоя кода.
- Постепенная раскатка (Canary releases).
- A/B тестирование.

## Инструкция по внедрению:
1. Определи условие флага.
2. Оберни новую логику в проверку флага.
3. Опиши, как этот флаг будет удален после полной раскатки (Code Cleanup).
