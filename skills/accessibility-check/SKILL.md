---

Используй этот скилл для проверки интерфейса на инклюзивность.

## Чек-лист:
1. **Семантика**: Используй `<nav>`, `<main>`, `<header>` вместо простых `<div>`.
2. **Атрибуты ARIA**: Проверяй наличие `aria-label`, `aria-expanded`, `role` там, где это необходимо.
3. **Контрастность**: Цвета текста и фона должны быть легко различимы.
4. **Клавиатура**: Убедись, что по всем интерактивным элементам можно перемещаться с помощью `Tab`.
5. **Изображения**: Все `<img>` должны иметь атрибут `alt`.

## Метод:
Проанализируй HTML/React компонент и предложи исправления для улучшения доступности.
