Скилл для создания "живых" интерфейсов.

## Техника:
1. **Производительность**: Анимируй только `transform` и `opacity` (они самые быстрые для браузера).
2. **Микровзаимодействия**: Плавные ховер-эффекты (`transition-all duration-300`), появление элементов при скролле.
3. **Keyframes**: Создавай уникальные анимации пульсации, плавного всплытия или "летающих" элементов.
4. **Easing**: Используй сложные кривые Безье (`cubic-bezier`), вместо обычного `linear` или `ease-in`.

## Пример:
Если пользователь хочет кнопку, не просто дай `bg-blue-500`. Сделай её с легким градиентом, который смещается при наведении, и мягким увеличением.
