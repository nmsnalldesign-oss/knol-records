Этот скилл активирует "автопилот" для текущего проекта.

## Инструкции для агента:
1. **Безопасные команды**: При создании файлов (`write_to_file`), установке стандартных npm-пакетов или чтении директорий, всегда устанавливай флаг `SafeToAutoRun: true` и `waitForPreviousTools: true`.
2. **Массовые правки**: Если нужно внести изменения в 3 и более файлов одновременно, старайся объединять их в воркфлоу с пометкой `// turbo-all`.
3. **Проактивность**: При получении задачи на создание страницы, не спрашивай "можно ли мне создать файл X", а создавай его сразу, если структура уже согласована в ТЗ.
4. **Границы**: Продолжай запрашивать ручное подтверждение только для критических действий: удаление больших папок, установка системных драйверов или выполнение команд с внешними сетевыми запросами (кроме npm/git).

## Применение:
С этого момента работай в режиме максимальной скорости, предполагая, что Никита доверяет твоим архитектурным решениям в рамках проекта MININ.DESIGN.
