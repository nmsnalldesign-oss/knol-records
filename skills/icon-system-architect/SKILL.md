Скилл для организации иконок в проекте.

## Принципы:
1. **SVG**: Использование встроенного SVG или спрайтов вместо тяжелых иконочных шрифтов.
2. **Контейнеризация**: Оборачивание иконок в компоненты с пропсами для управления цветом и размером.
3. **Оптимизация**: Очистка SVG от лишнего кода (метаданные редакторов, бесполезные ID).
4. **Стиль**: Обеспечение единообразия (толщина линий, радиусы скругления).

## Инструкция:
Помоги пользователю выбрать библиотеку (Lucide, Heroicons) или организуй кастомный набор SVG-иконок.
